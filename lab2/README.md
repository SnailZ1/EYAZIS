# PDF Language Recognition Suite

Приложение на CustomTkinter для автоматического определения русского и немецкого языков в PDF‑документах. Система объединяет классические методы обработки текста и нейросетевой детектор, предоставляя аналитические панели, карточки результатов и гибкие сценарии работы — от одиночных проверок до пакетной обработки.

## Основные возможности

- Переключаемые тёмная и светлая темы, drag‑and‑drop загрузка, индикатор прогресса и визуализации (гистограммы, heatmap, таймлайн).
- Три независимых метода распознавания:
  - **Частотный профиль** — сравнение топ‑100 слов с языковыми профилями и косинусное сходство.
  - **Метод коротких слов** — анализ вероятностей для слов ≤5 символов.
  - **Нейросетевой детектор** на базе `langdetect`.
- Пакетная обработка папок, режим сравнения, предпросмотр текста, экспорт в CSV/XLSX/PDF, история операций и уведомления.
- Обучающая вкладка для пересборки профилей прямо из пользовательских корпусов.

## Быстрый старт

1. **Установка зависимостей**

   ```bash
   pip install -r requirements.txt
   ```

2. **Запуск приложения**

   ```bash
   python main.py
   ```

3. **(Опционально) генерация демо‑PDF**

   ```bash
   python samples/generate_samples.py
   ```

## Примечания

- Профили `ru.json` и `de.json` содержат стартовые частотные данные; их можно пересоздать во вкладке «Обучение».
- Статистика и графики заполняются по мере накопления записей в `data/history.csv`.
- Все пути формируются через `pathlib`, поэтому проект одинаково работает на Windows и других платформах.
